# 🤖 Minecraft Bots Commander (Mod + Server)

Современная система удаленного управления внутриигровыми ботами в Minecraft. Проект состоит из двух связанных частей: **Java-мода** для клиента игры и **управляющего сервера** на Python с современным графическим интерфейсом.

---

## 📂 Структура репозитория

* `bots-commande/` — Исходный код модификации Minecraft (Fabric 1.21.8), которая принимает команды от сервера, имитирует действия игрока и взаимодействует с интерфейсами (инвентарь, аукцион(для Funtime)).
* `bots-commander-server/` — Асинхронный многопоточный сервер управления на Python 3.14 с графическим интерфейсом на базе **CustomTkinter**.

---

## 🛠️ Основной функционал

* **Многопоточный сервер:** Поддерживает одновременное подключение любого количества ботов без задержек и взаимной блокировки.
* **Управление инвентарем и GUI:** Автоматическое закрытие окон (`closeHandledScreen`) и предотвращение рассинхронизации с сервером.
* **Игнорирование ресурспаков:** Автоматическое отклонение серверных текстурпаков на этапе подключения через `ResourcePackPolicy.DISABLED`.
* **Удобная панель управления:** Полный набор заготовленных команд для автоматизации рутины:
    * `msg` — отправка сообщений в чат.
    * `command` — выполнение игровых команд (автоматический стрип знака `/`).
    * `changenik` — динамическое обновление никнейма в панели управления.
    * `connect` — быстрая смена игрового сервера.
    * `refreshah`, `takeallfromah`, `dropall` — автоматизация работы с аукционом и очистка инвентаря.
* **Гибкая маршрутизация:** Отправка как массовых команд (`-all`), так и точечных указаний конкретному боту (`-single`).
* **Локальная БД:** Сохранение списка ботов и их последнего сетевого статуса (online/offline) в JSON-файл.

## 🚀 Инструкция по запуску

### 1. Сервер управления (Python)
Убедитесь, что у вас установлен Python (протестировано на версиях 3.10+).

```bash
# Перейдите в папку с сервером
cd bots-commander-server

# Создайте и активируйте виртуальное окружение (для Linux/Arch)
python -m venv .venv
source .venv/bin/bin/activate

# Установите необходимые зависимости
pip install -r requirements.txt

# Запустите сервер с графическим интерфейсом
python servergui.py или просто python server.py
